  # INTRODUCTION  

A modern and responsive **User Management Dashboard** built using **HTML, CSS, and JavaScript**. This project allows users to perform CRUD operations with additional features like search, filter, sorting, pagination, and local storage support.

  # Features

*  View list of users
*  Add new users
*  Edit existing users
*  Delete users
*  Search users (by name & email)
*  Filter users (name, email, department)
*  Sort users (Name & Email - A/Z)
*  Pagination (custom rows per page)
*  Local Storage support (data persistence)
*  Reset data
*  Loader animation
*  Toast notifications
*  Fully responsive UI



 # Tech Stack

* **HTML5**
* **CSS3** (Modern UI with gradients & responsive design)
* **JavaScript (Vanilla JS)**
* **JSONPlaceholder API** (Mock backend)

 # How It Works

* Data is fetched from a mock API:

  ```
  https://jsonplaceholder.typicode.com/users
  ```

* Users can:

  * Modify data (Add/Edit/Delete)
  * Data is saved in **localStorage**
  * Combined with API data for persistence
